# Subscription URLs

Paste an `https://…` subscription link in the top bar and click **Import URL** (desktop) or **Import** (Android).

## Saved URL & refresh

After a successful import, the subscription URL is **saved** in your settings. Use **Refresh** to fetch the latest server list without pasting the link again.

New servers from a refresh are merged into your list; duplicates (same link, address, and port) are skipped.

## Formats

Subscriptions may be:

- Plain text, one share link per line
- Base64-encoded bundle of links
- A single `https://` URL that returns either format

Pasting an `https://` URL into the share-link box also triggers subscription import automatically.

See [Getting started](../GETTING_STARTED.md) for supported protocols.
