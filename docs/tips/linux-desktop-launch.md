# Linux launch

Extract the release zip and run:

```bash
chmod +x run-v2rayF.sh
./run-v2rayF.sh
```

The script sets the working directory next to `cores/` so Xray finds geo data and the binary. Keep the folder layout intact when moving installs.

Wayland and X11 are supported via Avalonia; use your distro's GPU drivers if the UI renders slowly.
